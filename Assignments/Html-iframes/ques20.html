Head Elements Explained

<title> - Specifies the page title shown in the browser tab.

<meta charset> - Declares the character encoding (UTF-8).

<meta name="description"> - Gives a short description used by search engines.

<meta name="author"> - Declares the author of the document.

<base> - Sets the base URL for resolving relative links.

<link> - Links to external resources like favicons or stylesheets.

<style> - Embeds CSS rules directly in the document.

<script> - Embeds JavaScript in the document.
